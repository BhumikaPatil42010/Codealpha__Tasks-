Stock Portfolio Tracker
*  Project Overview

The Stock Portfolio Tracker is a Python-based application that helps users calculate and manage their stock investments. The program allows users to add stocks, specify quantities, calculate total investment value, and save portfolio details into a CSV file.

This project demonstrates how basic financial tracking tasks can be automated using Python programming.

*  Objective

The main objective of this project is to:

Track stock investments efficiently

Calculate total investment value

Practice core Python programming concepts

Implement file handling for data storage

*  Working of the Project

The program displays a menu with available options.

Users can add stocks by entering stock name and quantity.

Stock prices are retrieved from a predefined dictionary.

The system calculates total investment automatically.

Portfolio details can be saved into a CSV file.

The program continues until the user chooses to exit.
