Hangman Game – Python Project
* Project Overview

The Hangman Game is a simple text-based Python game where the player guesses a hidden word one letter at a time. The game randomly selects a word from a predefined list, and the player must guess the word before reaching the maximum number of incorrect attempts.

This project was developed as part of an Internship Major Project to demonstrate basic Python programming concepts and logical problem-solving skills.

* Objective

To develop a console-based Hangman game using Python.

To apply fundamental programming concepts such as loops, conditions, lists, and strings.

To create an interactive user experience using ASCII graphics.

* Technologies Used

Programming Language: Python

Platform: VS Code / Google Colab / Command Prompt

* Conclusion

This project helped in understanding Python fundamentals, logical thinking, and game development concepts. The Hangman Game demonstrates how basic programming constructs can be combined to create an interactive application.
